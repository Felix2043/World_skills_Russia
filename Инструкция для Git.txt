Первоначальная настройка git можно пропустить
git config --global user.name "Название компьютера + номер"
git config --global user.email почта


Создать папку на рабочем столе


Переход в эту папку
cd Desktop/ Название папки

Клонирование репозитория
git clone Основной
https://try.gogs.io/GrandDemoResrv/DemoExam16.git


Переход в папку репозитория
cd Название папки

Инициализировать репозиторий
git init

Создание новой ветки:
git checkout -b имя_ветки


Закинуть все нужные файлы в папку с репозиторием


Добавить все файлы в репозитрий
git add .

Закоммитить изменения
git commit -m "Название коммита"

Запушить всё в ветку
git push https://try.gogs.io/GrandDemoResrv/DemoExam16.git имя_ветки
